# Setup
## Windows Setup
### Init
download XLaunch (GUI client)
1. wsl
2. cd padm-project
3. curl -s https://gitlab.com/mit-mers/teaching/padm-project-2022f/-/raw/main/install.sh | bash
4. export DISPLAY=:0
5. source ./.venv/bin/activate && cd padm-project-2022f/ && python minimal_example.py
### Subsequent Launches
1. wsl
2. cd padm-project
3. ./.venv/bin/activate
